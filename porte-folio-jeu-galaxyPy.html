Portefolio

Jeu Galaxy

Ce projet faisait partie de ma formation Python. Le but était de développer une application Python incluant une interface graphique dynamique à l'aide du Framework Kivy, et des interactions avec l'utilisateur.

Cette application est un jeu multiplateforme (smartphone, tablette ou ordinateur) en perspective où vous devez contrôler les directions d'un vaisseau et le faire avancer sur un chemin infini en constante création, le but du jeu étant d'avancer le plus loin possible sans sortir du chemin. Plus vous avancez dans le jeu, plus le vaisseau accélère.

Ce projet a permis d'apprendre à utiliser le Framework Kivy pour Python et d'aborder les notions de perspective, de design, de gestion de collisions et d'interactions avec l'utilisateur par appui-touche, sur ordinateur, et tactile sur smartphone et tablette.
