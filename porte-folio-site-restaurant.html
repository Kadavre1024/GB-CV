Pizza Mama

J'ai réalisé le projet "Pizza Mama" au cours de ma formation Python / Django. Le but était de se mettre en conditions réelles de travail et de développer, dans un premier temps, un site web constité d'une page d'accueil et d'une page "Menu". Dans un second temps, j'ai développé la partie base de données pour y inclure la liste des pizzas, et créé les interactions entre le site et la base avec Django, pour finir par y inclure une API REST. Cela permettant de passer à la dernière phase du projet, créer une application venant consommer l'API et afficher le menu.
